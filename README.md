## Assignment Overview
This project is an iOS product management app designed with Core Data.

## Required Features
- Store at least 10 products
- Display the first product on app launch
- Navigate through products
- Search by product name or description
- Add a new product
- Display a full product list with product name and description

## Product Fields
- Product ID
- Product Name
- Product Description
- Product Price
- Product Provider
