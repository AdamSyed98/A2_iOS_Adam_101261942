# Project Notes

## Planned Screens
1. Product Detail Screen
2. Add Product Screen
3. Product List Screen

## Core Data Entity
Entity Name: Product

Attributes:
- productID: String
- productName: String
- productDesc: String
- productPrice: Double
- productProvider: String

## App Flow
- App launches and loads default products if database is empty
- First product is displayed
- User can go to next/previous product
- User can search by product name or description
- User can add a product
- User can view all products in a list
